Creating a Menu using Project CX

We already know how to add different elements on our webpage. It's the right time to introduce the usage of Project CX Menus.

One can create their own menus with multiple menu items in no time when using Project CX.

  1. Go to the Admin Module
  2. Select the Admin Module Menu
  3. Select 'Page Management'
  4. Select 'Setup Menus'

  1. Now click on 'Add' button to create a new menu for your website
  2. Give your menu a name and add as many menu items as you want

  1. You can also add sub menus for a menu item by simply clicking the icon next to the menu item
  2. From there, you can add sub menu items as shown

  1. In a similar way, you can also add URIs or redirect the users to a page when they click on the menu item

  1. With all this in place, we now have a menu setup on our admin panel but we still need to add it to a webpage and style it.
