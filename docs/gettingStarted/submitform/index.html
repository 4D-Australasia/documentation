Submitted Forms

We are now ready to submit a form and see the result in the Admin Module.

  1. Open a browser and load the page - https://yourdomain/test
  2. Enter a First Name and Last Name into the fields - For Example, John Smith

We will now look at the data saved by Project CX.

  1. Press the Submit button
  2. Go to the Admin Module
  3. Select the Admin Module Menu
  4. Select 'Page Management'
  5. Select 'Submitted Forms'
  6. Select the Test form from the 'Form' dropdown to show the submitted form
  7. Select the form to open the row to view the submitted form data

We have now created our first page, added a form and can receive data from it.
